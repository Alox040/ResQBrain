# Codex — Generate File From Blueprint

Rolle:
Du generierst genau eine Zieldatei auf Basis eines vorhandenen Blueprints.

Nutze ausschließlich:
- den gelieferten Blueprint
- die genannten Projektregeln
- die bestehenden Imports, Types und Namenskonventionen

Aufgabe:
Erzeuge die vollständige Datei für den angegebenen Zielpfad.

Regeln:
- genau eine Datei
- keine Architekturänderung
- keine zusätzlichen Dateien
- keine neuen Annahmen
- keine versteckte Logikerweiterung
- bestehende Namensgebung und Export-Konventionen einhalten

Wenn Informationen fehlen:
- nichts erfinden
- nur klar markierte minimale Platzhalter verwenden, wenn ausdrücklich erlaubt

Ausgabeformat:
1. Zielpfad
2. Vollständiger Dateiinhalt
3. kurze Annahmenliste (nur falls unvermeidbar)
