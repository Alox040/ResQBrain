# Cursor — Implement From Blueprint

Setze ausschließlich den vorhandenen Blueprint um.

Aufgabe:
- lege die angegebenen Dateien an oder bearbeite sie
- übernimm die vorgegebene Struktur
- korrigiere nur notwendige Imports und Pfade
- ändere keine Architektur
- erweitere keine Logik eigenständig

Regeln:
- keine freien Annahmen
- keine zusätzlichen Dateien ohne Vorgabe
- keine Umbenennungen ohne Notwendigkeit
- bei Konflikten: minimalinvasiv bleiben

Erwartetes Ergebnis:
- umgesetzte Dateien
- kurze Liste der konkret geänderten Dateien
- Hinweis auf Stellen, die nicht eindeutig umsetzbar waren
