# Current Phase

**Last Updated:** 2026-03-27

## Aktuelle Phase: Phase 0

## Begruendung

Der aktuell sichtbare Implementierungsstand entspricht Phase 0 (Lookup App), weil:

- Mobile UI konzentriert sich auf Suche, Listen und Detailansichten fuer Medikamente/Algorithmen
- Datenquellen in der App sind statisch/mock-basiert
- keine produktive API-, Auth- oder Multi-Tenant-Runtime vorhanden
- keine aktive Lernlogik oder KI-Assistenz integriert

Parallel existiert bereits ein fortgeschrittenes Domain-Zielbild fuer spaetere Phasen (Governance, Versioning, Release, Tenant-Isolation).  
Diese Teile sind als Plattform-Fundament vorhanden, aber nicht als vollstaendiger End-to-End-Produktbetrieb in App + Backend umgesetzt.
