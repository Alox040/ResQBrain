Quelle: Subreddit Umfrage

---

# Kernerkenntnisse

## 1. Hauptproblem
Einsätze werden aktuell kaum systematisch nachbereitet.

Gründe:
- Zeitmangel
- zu komplizierte Dokumentation
- kein geeignetes Tool
- kein direkter Mehrwert sichtbar
- Lernen erfolgt unsystematisch

---

## 2. Stärkster Wunsch
Extrem schnelle Nachbereitung direkt nach Einsatz

Anforderungen:
- < 30 Sekunden
- minimaler Input
- keine langen Texte
- mobil nutzbar
- optional Spracheingabe

---

## 3. Lernbedarf
User wollen vor allem:

- aus eigenen Einsätzen lernen
- Muster erkennen
- Fehler reflektieren
- Entscheidungen nachvollziehen
- Wissen langfristig behalten

Nicht wichtig:
- klassische Theorie
- lange Lehrtexte
- umfangreiche Kurse

---

## 4. Informationsbedarf im Alltag

Hohe Priorität:

- schnelle Suche
- kurze Antworten
- kompakte Entscheidungshilfen
- Einsatzrelevantes Wissen
- sofort verfügbar

---

## 5. UX Anforderungen

Sehr wichtig:

- extrem schnell
- minimalistisch
- offlinefähig
- keine Pflichtfelder
- optional später erweitern

---

# Interpretation

User wollen KEIN:

- Dokumentationssystem
- Lernplattform
- Wissensdatenbank

User wollen:

→ "Einsatz merken → automatisch lernen"

---

# Produkt Richtung

Primär:
- Einsatznachbereitung

Sekundär:
- Lernen
- Wissensaufbau

Tertiär:
- Entscheidungsunterstützung

---

# Wichtigste MVP Features

1. Schnell-Nachbereitung
2. Einsatz markieren
3. kurze Stichworte
4. optional Spracheingabe
5. automatische Lernkarten
6. später erneut anzeigen
7. Suche
