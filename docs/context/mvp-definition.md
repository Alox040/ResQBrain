# MVP Definition basierend auf Umfrage

Ziel:
Minimal funktionierender Kern für praxisnahe Nutzung im Alltag

---

# MVP Core

## Feature 1 — Schnell-Nachbereitung

User kann nach Einsatz:

- Einsatz markieren
- Kategorie wählen
- optional Stichwort
- optional Spracheingabe

Zeit: < 30 Sekunden

---

## Feature 2 — Lern-Automatik

System erzeugt:

- Lernpunkte
- Wiederholungen
- Erinnerung
- Muster

---

## Feature 3 — Einsatzliste

User sieht:

- letzte Einsätze
- markierte Einsätze
- Lernpunkte

---

## Feature 4 — Suche

User kann suchen:

- Thema
- Medikament
- Entscheidung
- Einsatzart

---

# MVP UX Regeln

Keine Pflichtfelder  
Keine langen Formulare  
Keine Navigationstiefe  
Direkt nach Einsatz nutzbar  

---

# NICHT im MVP

keine KI Analyse  
keine Organisation Features  
keine Export Funktionen  
keine Rollenverwaltung  
keine komplexe Struktur  

---

# MVP Ziel

Nachbereitung wird wirklich genutzt
