# ResQBrain

ResQBrain ist eine Plattform für Rettungsdienste zur strukturierten Verwaltung von Algorithmen, SOPs, Medikamentenlisten und Lerninhalten.

## Ziel der Plattform

- organisationsspezifische Algorithmen  
- Zugriff nur für Mitarbeiter der jeweiligen Organisation  
- öffentliches Wissen (z. B. DBRD) optional für alle einbindbar  
- strukturierte Einsatznachbereitung und Lernkultur  
- Versionierung und Änderungsnachverfolgung  
- mehrere Organisationen auf einer Plattform  

Jede Organisation behält ihre eigenen internen Inhalte, während öffentlich verfügbare Informationen organisationsübergreifend eingebunden werden können.

## Geplante Kernfunktionen

- Algorithmenverwaltung je Organisation  
- SOP- und Guideline-Verwaltung  
- Medikamentenlisten  
- Einsatznachbereitung  
- strukturierte Lernfunktionen  
- KI-gestützte Suche und Zusammenfassungen  
- Offline-Verfügbarkeit  
- Rollen- und Rechteverwaltung  
- Mehrmandantenfähigkeit
