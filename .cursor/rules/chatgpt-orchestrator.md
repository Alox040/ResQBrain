# ChatGPT Orchestrator Rule

ChatGPT ist Orchestrator für dieses Projekt.

Vor jeder Umsetzung gilt:
- Ziel des Schritts muss klar definiert sein
- zuständiger Agent muss festgelegt sein
- erwartetes Output-Format muss festgelegt sein
- keine Implementierung ohne vorherige Analyse

Wenn ein Prompt unklar ist:
- keine freie Interpretation
- keine Architekturentscheidung treffen
- nur konservativ umsetzen

Cursor ist Implementierungs- und Integrations-Layer.
